4--> What is an URL?



Ans-->URL is the abbreviation of Uniform Resource Locator. It is the global address of documents and other resources on the World Wide Web. For example, www.webopedia.com is a URL. A URL is one type of Uniform Resource Identifier (URI); the generic term for all types of names and addresses that refer to objects on the World Wide Web.
Parts of a URL-:
The first part of the URL is called a protocol identifier and it indicates what protocol to use, and the second part is called a resource name and it specifies the IP address or the domain name where the resource is located. The protocol identifier and the resource name are separated by a colon and two forward slashes.
